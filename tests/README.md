# TESTS

## Objetivo

Validar estrutura, configuração e integridade mínima do projeto.

## Execução

- `npm test`

## Cobertura mínima

- Arquivos obrigatórios existem e têm conteúdo.
- `config.json` contém seções obrigatórias.
- Seeds essenciais estão presentes.
- O entrypoint web existe.

## Critério

- Falha em qualquer verificação = projeto não validado.
