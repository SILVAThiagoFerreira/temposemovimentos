# OUTPUT

Saídas geradas pelo sistema.

## Tipos

- CSV de apontamentos.
- JSON de backup completo.
- Relatórios de estado via UI.

## Convenções

- Nomes de arquivo seguem `config.json`.
- CSV exporta somente o recorte filtrado na tela.
- Backup JSON exporta a base completa.

## Observação

- Este diretório documenta o padrão de saída. A geração real ocorre na interface web.
